## egg_travel

- 创建项目`yarn create egg --type=simple`
- 配合前端项目[umi3.x_travel](https://github.com/xmllein/umi3.x_travel) `no-mock-branch` 分支

## 项目运行

- 配置 `mysql`, `redis`
- 项目中`app.sql` 导入数据

## 主要功能

- [x] 用户登录注册，编辑用户 接口
- [x] 首页搜索，热门民宿 接口
- [x] 搜索结果 接口
- [x] 民宿详情 接口
- [x] 民宿评论 接口
- [x] 名宿预定，名宿取消订单， 民宿订单，模拟支付 接口
- [ ] docker 部署
